НЕМНОГО О ПРОЕКТЕ:
  Целью работы стала разработка нейросети, способной диагностировать вероятные заболевания. Я выбрал это направление, чтобы реализовать возможность использования искусственного интеллекта как рабочий инструмент. Проект решает несколько задач: он помогает анализировать комбинации симптомов, сужать круг возможных заболеваний.
  В современных условиях растет нагрузка на медицинских специалистов, что делает невозможным проведение глубокого первичного осмотра каждому пациенту сразу же.
  Традиционные методы самостоятельной диагностики через поисковые системы часто дают противоречивую или пугающую информацию из-за сложной медицинской терминологии. Именно поэтому моя работа направлена на решение этой проблемы.
  В ходе работы была создана полнофункциональная система. Технически модель представляет собой нейронную сеть с 5 скрытыми слоями и регуляризацией, что позволяет ей избегать переобучения. Она обучалась на наборе данных объемом 5000 записей, включающих информацию о почти 40 различных заболеваниях и множестве симптомов.
  Реализован удобный поиск и фильтрация симптомов по категориям («Общие», «Кожные», «Неврологические» и др.). Для удобства пользователя добавлены «Быстрые примеры» (Простуда, Диабет, Аллергия), позволяющие мгновенно увидеть работу алгоритма без ручного ввода.
  В верхней панели размещен логотип «AI Диагностика» и описание функционала системы.
  Реализован удобный поиск и фильтры по категориям — «Общие», «Кожные», «Неврологические», «ЖКТ», «Дыхательные». Сами симптомы представлены в виде кнопок в виде сетки, что позволяет быстро выбрать нужные. Также есть счётчик «Выбрано: 0», который показывает пользователю прогресс.
  Большая яркая кнопка «Диагностировать» призывает пользователя завершить ввод данных, а соседняя кнопка «Очистить» позволяет начать заново.
  Для демонстрации работы модели добавлены готовые карточки с примерами заболеваний (например, Простуда, Диабет), чтобы пользователь мог сразу проверить точность алгоритма.
  При получении ответа система отображает вероятные диагнозы с процентом совпадения в виде цветных полосок (зелёный для высокой вероятности, жёлтый и серый для остальных). Ниже приводятся рекомендации по лечению и профилактике.
  Результат работы выходит за рамки простого диагноза. Система выдает вероятность заболевания, а затем формирует базу знаний. Это включает рекомендации по лечению и советы по профилактике, что полностью соответствует заявленной цели проекта по помощи пользователю.
Заключение:
Итогом проектной работы стало создание веб-сайта с исходным кодом, презентацией и обученной нейросетью. Точность классификации составила от 85% до 92%, задачи выполнены полностью, проект готов к использованию.
